# Grid System

## How it works

The grid system uses a series of containers, rows, and columns to layout and align content.
It's built with [flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout/Using_CSS_flexible_boxes)
and is fully responsive.

We use a 12 column grid system driven by [Bootstrap 5.2](https://getbootstrap.com/docs/5.2/layout/grid/) under the hood.

Here's how the grid changes across breakpoints:

## Auto Layout Columns

Utilize breakpoint-specific columns for easy column sizing without an explicit size.

### Equal-width

Here are two grid layouts that apply to every device and viewport, from `xs` to `xl`.
Add any number of unit-less columns for each breakpoint you need and every column
will be the same width.

### Setting one column width

Auto-layout for flexbox grid columns also means you can set the width of one
column and have the sibling columns automatically resize around it.

Note that the other columns will resize no matter the width of the center column.

### Mix and match

Don't want your columns to simply stack in some grid tiers?
Use a combination of different classes for each tier as needed.

<Canvas>
  <Story name="Mix and match">
    <Grid>
      {/* Stack the columns on mobile by making one full-width and the other half-width */}
      <Grid.Row>
        <Grid.Col md={8}>md=8</Grid.Col>
        <Grid.Col size={6} md={4}>size=6 md=4</Grid.Col>
      </Grid.Row>
      {/* Columns start at 50% wide on mobile and bump up to 33.3% wide on desktop */}
      <Grid.Row>
        <Grid.Col size={6} md={4}>size=6 md=4</Grid.Col>
        <Grid.Col size={6} md={4}>size=6 md=4</Grid.Col>
        <Grid.Col size={6} md={4}>size=6 md=4</Grid.Col>
      </Grid.Row>
      {/* Columns are always 50% wide, on mobile and desktop */}
      <Grid.Row>
        <Grid.Col size={6}>size=6</Grid.Col>
        <Grid.Col size={6}>size=6</Grid.Col>
      </Grid.Row>
    </Grid>
  </Story>
</Canvas>
